# 交通违法举报小程序 - 测试验证教程

## 准备工作

### 1. 安装微信开发者工具

下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 获取 AppID

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发」→「开发管理」→「开发设置」
3. 记录你的 AppID（或使用测试号）

---

## 项目导入

### 1. 打开项目

1. 启动微信开发者工具
2. 点击「导入项目」
3. 选择目录：`traffic_vigilante/miniprogram`
4. 填入 AppID
5. 点击「确认」

### 2. 目录结构确认

```
traffic_vigilante/
├── miniprogram/
│   ├── app.js              # 应用入口
│   ├── app.json            # 全局配置
│   ├── app.wxss            # 全局样式
│   ├── mock/
│   │   └── mock-data.js    # 模拟车机数据
│   ├── utils/
│   │   └── report-generator.js  # 举报包生成工具
│   ├── components/
│   │   ├── video-timeline/
│   │   └── violation-type-picker/
│   └── pages/
│       ├── index/          # 首页
│       ├── scan/           # 扫码页
│       ├── clip/           # 裁剪页
│       ├── confirm/        # 确认页
│       └── result/         # 结果页
```

## 功能测试流程

### Step 1: 首页 - 加载模拟数据

1. 进入小程序，首页显示「交通违法举报」标题
2. 点击「使用模拟数据」按钮
3. 页面显示「已接收 2 个标记点」
4. 点击「开始裁剪视频」进入裁剪页

**预期结果：**
- Toast 提示「已加载模拟数据」
- 标记点数量显示为 2
- 「开始裁剪视频」按钮可点击

### Step 2: 裁剪页 - 标记点导航

1. 页面加载后显示第一个标记点
2. 显示道路名称：「京沪高速上海方向」
3. 显示时间戳
4. 显示裁剪参数控制条

**操作测试：**
- 点击「上一个」→ 按钮禁用（已是第一个）
- 点击「下一个」→ 切换到第二个标记点
- 再次点击「下一个」→ 按钮禁用（已是最后一个）

### Step 3: 裁剪页 - 调整裁剪参数

1. 拖动「标记前缓冲」slider
   - 从 3 秒调整到 5 秒
   - 观察 clipStart 自动变化

2. 拖动「裁剪开始」slider
   - 观察数值实时变化

3. 拖动「裁剪结束」slider
   - 观察数值实时变化

**预期结果：**
- 各 slider 响应灵敏
- 数值实时更新显示

### Step 4: 确认页 - 违法类型选择

1. 在裁剪页点击「确认裁剪」
2. 进入确认页，显示：
   - 道路名称（可编辑）
   - 违法时间
   - 裁剪范围（格式：01:05 - 02:05）

**操作测试：**
- 点击违法类型选项（如「非法变道」）
- 选项高亮显示
- 不选择违法类型，直接点击「生成举报包」→ 提示「请选择违法类型」

### Step 5: 结果页 - 举报信息

1. 选择违法类型后点击「生成举报包」
2. 进入结果页，显示：
   - 绿色✓图标
   - 「举报包已生成」
   - 举报编号
   - 违法类型
   - 违法地点

**操作测试：**
- 点击「打开12123上传」
- 检查剪贴板是否复制了举报信息
- 弹出提示框引导打开12123 App

## 扫码功能测试（需配合车机）

如需测试扫码功能：

1. 在车机端生成包含以下格式的二维码：
```json
{
  "marks": [
    {
      "timestamp": "2026-04-20T10:30:05",
      "gps": { "lat": 31.2304, "lng": 121.4737 },
      "roadName": "京沪高速上海方向",
      "videoFile": "record_001.mp4",
      "offset": 5
    }
  ]
}
```

2. 在小程序首页点击「扫码接收」
3. 扫描二维码
4. 验证数据是否正确接收

## 常见问题

### Q: 点击按钮没有反应
A: 检查是否有 Toast 提示，检查控制台是否有报错

### Q: 裁剪页视频不播放
A: 视频使用本地路径（`/record_001.mp4`），实际需替换为车机热点可访问的视频地址

### Q: 举报包生成失败
A: 确认已选择违法类型且道路名称不为空

### Q: 无法跳转到12123
A: 结果页会将举报信息复制到剪贴板，需手动打开12123 App粘贴上传
